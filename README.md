# Data Cleaning Project

## 📌 Project Overview
This project focuses on cleaning and preprocessing two datasets:
- Airbnb Dataset

- Removed outliers

1. Airbnb Dataset
2. YouTube Trending Dataset

## 📊 Output
- cleaned_airbnb.csv
- cleaned_youtube.csv

## 🛠 Tools Used
- Python
- Pandas

## 🎯 Conclusion
Data cleaning improved the quality and reliability of datasets for further analysis.
